# 꾹꾹이

> 종이 쿠폰을 대체하는 간편한 전자 스탬프 플랫폼
- 꾹꾹이를 사용하는 모든 카페에서 쉽고 간편하게 스탬프를 **적립**하는 서비스
- 마시고 싶은 음료를 **검색**해 가까운 매장을 알려주는 위치 기반 서비스
- 적립한 스탬프를 친구에게 **선물**하는 스탬프 이체 금융 서비스
